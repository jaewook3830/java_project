컴파일 실행 : javac 파일이름.java
인터프리터 실행 : java 파일이름

=======================================================================

명명법(식별자) : 이름 만드는 방법
1.클래스 : 첫글자는 대문자 나머지는 소문자
	 보통 명사를 사용, 숫자는 중간이나 끝에 사용가능
	 두단어 이상일때는 단어의 첫글자들은 무조건 대문자를 사용한다
 객체를 생성하기 위한 자원을 클래스이다. (클래스를 이용해서 객체를 만든다.)
 객체 : 사용할 수 있는 자원
 파일로 저장할 수 있는 단위 (클래스이름이 저장이름)  
 자바 API는 클래스들로 이루어져 있다.	 

2.메소드 : 클래스 안에 존재한다.
	 메소드이름() 형태이다.
	 동작, 기능을 의미한다.
	 첫글자는 소문자이다.
	 두 단어이상일때 두번째 단어부터 첫글자는 대문자이다.
	 javaTestExam

3.변수와 상수 : 클래스나 메소드 안에 존재한다.
  1) 변수 : 첫글자는 소문자를 사용
	 두 단어 이상 일때 두번째 단어부터 첫글자는 대문자이다.
	 (명명법은 메소드와 같다)
  2) 상수 : 모든 글자가 대문자로 이루어져 있다.
	 보통 단어와 단어사이에 under score( _ )를 사용한다. ( JAVA_HOME, JAVAHOME )
  
  변수와 상수
  변수 : 프로그램의 시작에서 끝날때까지 언제든지 가지고 있는 데이터가 변할 수 있는 저장공간을 의미
          가장 최신의 정보 하나만을 기억할 수 있는 저장공간이다.
  상수 : 프로그램의 시작에서 끝날때까지 한번 지정한 데이터는 변할 수 없는 저장공간을 의미

  자바에서 변수와 상수를 표현하는 방법
  변수 :  
	자료형 변수이름;       // 변수 선언 ( 한번만 하면 된다 )
	변수이름 = 데이터;    // 변수에 데이터를 저장 ( 절대로 같다라는 의미가 아님 )

      	자료형 변수이름 = 데이터;
  상수 : static final 자료형 상수이름;
          final 자료형 상수이름;
